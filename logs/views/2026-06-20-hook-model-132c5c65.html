Learning Session - Hook Model 上瘾模型

Date: 2026-06-20 Topic: Hook Model / 上瘾模型

用户原始问题

什么是 HOOK model？请简单易懂解释并举事例。

通俗解释

Hook Model 是一个解释“用户为什么会反复回来使用某个产品或内容”的模型。它包括四步：触发、行动、多变酬赏、投入。

科研例子

在图书馆短视频研究中，可以用它分析一条短视频如何让用户从“看到标题”到“点开观看”，再到“获得知识或情绪回报”，最后到“关注、收藏、报名活动或继续使用图书馆服务”。

关键概念

  • Trigger 触发
  • Action 行动
  • Variable Reward 多变酬赏
  • Investment 投入

易错点

  • 不要把 Hook Model 简单理解成操纵用户上瘾。
  • 不要把点赞、评论、转发直接等同于图书馆服务价值。
  • 不要把基于 Hook Model 的相关性分析写成因果结论。

需要创建/更新的知识卡片

  • Hook Model 上瘾模型

复习任务

  • 用自己的话解释 Hook Model 四步。
  • 给一个图书馆短视频例子，分别指出触发、行动、酬赏和投入。
