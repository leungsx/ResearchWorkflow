# Happy Research Loop Prompt

请把我当前的灵感、问题、数据、实验结果或论文段落，放入“快乐科研闭环”中处理。

目标：

1. 先判断我现在处在哪一环：灵感、想法、研究问题、假设、实验、结果、主张、写作、归档。
2. 不要求我手工整理文件。
3. 用最少的问题推进一个小闭环。
4. 把结果沉淀到对应文件：idea card、hypothesis registry、experiment plan、result interpretation、claim-evidence map。
5. 最后告诉我下一步最轻松但最有价值的动作。

输出：

- 当前环节
- 需要我确认的问题
- Codex 将写入或更新的文件
- 下一步小任务
