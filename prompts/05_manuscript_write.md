# 中文论文写作 Prompt

默认目标期刊为《图书情报工作》。请基于研究问题、文献矩阵、实验结果、图表说明、目标期刊 profile 和引用库，帮助我写中文论文草稿。

要求：

1. 先判断文章类型：专题研究 / 理论研究 / 工作研究 / 情报研究 / 知识组织 / 综述述评 / 海外观察。
2. 中文摘要按 `[目的/意义]`、`[方法/过程]`、`[结果/结论]` 组织。
3. 正文围绕信息资源管理、图书馆、情报、知识服务、数据资源或智能服务等图情语境展开。
4. 每个实质性论断都标注 citekey、数据、实验结果或图表来源。
5. 不新增未提供的文献，不虚构数据、案例、访谈、问卷、统计结果或政策文件。
6. 明确区分结果、解释、推论、建议和局限。
7. 主动标出证据薄弱段落和投稿前必须补充的材料。
8. 准备长英文摘要的要点：Purpose/Significance、Method/Process、Result/Conclusion、Innovation/Value、Insufficient/Improvement。
9. 记录数据可用性和 AI 使用说明的草稿。
10. 输出 Markdown，引用使用 Pandoc citeproc 格式。
