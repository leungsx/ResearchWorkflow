30 秒定位

  • 本项目角色：图书馆短视频相关研究 的理论框架补位文献。
  • 为什么今天读：它是现有 8 篇学习集里最后一个还没稳定沉淀进综述和知识图谱的 Reader。
  • 它补的不是“内容技巧”，而是用户从感知到行动、再到分享的完整路径。

核心内容

  • 研究问题：公共图书馆短视频账号为什么整体传播力偏弱，以及如何沿着 SICAS 五阶段改进营销与服务转化。
  • 理论/概念：SICAS模型、沉浸式内容、软营销、用户感知价值、精神激励。
  • 方法与数据：
  • 在抖音以“图书馆”为关键词搜用户名，人工筛选 11 个粉丝总量过万的公共图书馆账号。
  • 结合飞瓜数据与手工收集数据。
  • 用 DCI传播力指数 从发布、互动、覆盖三维度描述传播影响力。
  • 主要发现：
  • 头部公共图书馆账号整体传播影响力仍偏弱。
  • 问题不只在内容，也在私域布局、平台外引流、主页链接、评论互动和分享意愿。
  • 作者据此提出“融媒体感知 -> 沉浸式兴趣 -> 互动连接 -> 软营销行动 -> 激励分享”的策略链条。

证据和边界

  • 可用证据 blocks：
  • B0005-B0006：SICAS 模型与公共图书馆短视频营销模型。
  • B0007-B0011：11 个账号的 DCI、传播诊断与五阶段问题。
  • B0011-B0016：五类策略建议与结语。
  • 证据边界：
  • 当前为 source-grounded skim，页码待核原 PDF。
  • 样本是 2021 年末的头部公共图书馆抖音号，不代表全部图书馆，也不代表 2026 平台状态。
  • DCI 与互动表现不能直接等于阅读推广质量、服务触达或公共文化价值。

创新 / 局限 / 机会

创新

  • 把 SICAS模型 从一般营销行为路径转译到公共图书馆短视频服务场景。
  • 把“感知渠道、互动连接、行动触发、分享回流”一起纳入，而不是只讨论内容特征。
  • 为 Hook Model 上瘾模型、ELM 精细加工可能性模型、AARRR 漏斗模型 之间提供一条更完整的过程框架。

局限

  • 偏描述性诊断和框架转译，缺少严格效果检验。
  • 互动与传播指标仍然替代了真实服务结果。
  • 样本和时间窗口都较窄，外推边界清楚。

机会

  • 图书馆短视频服务价值框架：把平台互动、服务触达、阅读转化纳入同一评价体系。
  • SICAS-服务转化路径：比较 SICAS 与 Hook/ELM/AARRR 在图书馆场景下的互补关系。
  • 图书馆短视频多平台复测：用 2024-2026 数据检验这套路径是否仍成立。

与已读文献的关系

  • 与 Hook Model 上瘾模型：Hook 更适合解释单条视频的微观循环；SICAS 更适合解释多触点路径与回流。
  • 与 ELM 精细加工可能性模型：ELM 强在解释线索加工；SICAS 强在解释从感知到行动的阶段连接。
  • 与 基于短视频营销的公共图书馆数字阅读推广策略研究：张承 2021 给服务设计方向，这篇给过程结构。
  • 与 公共图书馆和高校图书馆短视频营销比较研究：AARRR 更适合指标组织，SICAS 更适合过程建模。
