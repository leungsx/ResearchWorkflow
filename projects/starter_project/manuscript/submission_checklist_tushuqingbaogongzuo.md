# 《图书情报工作》投稿检查清单

Use this checklist before submission or internal pre-review.

## 1. Topic And Article Type

- [ ] 文章类型已明确：专题研究 / 理论研究 / 工作研究 / 情报研究 / 知识组织 / 综述述评 / 海外观察。
- [ ] 研究问题属于信息资源管理或相关领域。
- [ ] 已说明理论创新、方法创新、实践价值或政策意义。
- [ ] 与当前重点选题或期刊范围的关系已写入 `manuscript/target_journal.md`。

## 2. Chinese Manuscript Structure

- [ ] 中文题目清楚，避免口号化。
- [ ] 作者、单位、城市、邮编完整。
- [ ] 中文摘要包含 `[目的/意义]`、`[方法/过程]`、`[结果/结论]`。
- [ ] 中文关键词 3-8 个。
- [ ] 中图分类号已填写。
- [ ] 正文标题层级清楚，一级/二级/三级标题有对应英文。
- [ ] 基金项目脚注不超过两个项目。
- [ ] 作者简介包含第一作者或通信作者 E-mail。

## 3. English Components

- [ ] 英文题目与中文题目对应。
- [ ] 作者姓名汉语拼音、单位英文信息完整。
- [ ] Long English abstract 为 400-800 English words。
- [ ] English abstract 包含 Purpose/Significance、Method/Process、Result/Conclusion、Innovation/Value、Insufficient/Improvement。
- [ ] English keywords 与中文关键词对应。

## 4. Evidence And Integrity

- [ ] 每个核心论断能追溯到文献、数据、实验结果或图表。
- [ ] `07_claim_evidence_map.md` 已更新。
- [ ] `passport/material_passport.json` 已生成且覆盖 manuscript、data、analysis、figures。
- [ ] 没有把 AI 生成文本、图、代码或分析包装成人工原创。
- [ ] AI 使用过程、工具、作用和贡献已记录，必要时写入正文相关部分。

## 5. Data And Methods

- [ ] 数据来源、采集范围、样本、时间窗口或案例选择清楚。
- [ ] 数据处理步骤可复现。
- [ ] 如有可共享数据，已准备数据可用性声明、链接、DOI/CSTR 或仓储信息。
- [ ] 如不宜共享，已说明伦理、隐私、保密、第三方权益或无新增数据等理由。
- [ ] 软件、算法、模型、协议或材料已纳入复现记录。

## 6. Figures, Tables, And Maps

- [ ] 每张最终图有 `figures/specs/*.md` 图件合同。
- [ ] 图表题名均有中文和英文。
- [ ] 表格表头不为空；必要时使用三线表。
- [ ] 图中文字清晰，分辨率满足投稿级要求。
- [ ] 地图使用合规底图，并完成地图合规说明。
- [ ] 图表结论没有超过数据支持范围。

## 7. References

- [ ] 参考文献按 GB/T 7714-2015。
- [ ] 中文参考文献已补充英文翻译信息。
- [ ] 多作者文献的作者著录符合期刊要求。
- [ ] 数据集引用格式已检查。
- [ ] 引用不是“主题相关”，而是真正支持对应论断。

## 8. Authorship And Submission

- [ ] 第一作者和通信作者角色清楚。
- [ ] 未使用共同第一作者或共同通信作者表述。
- [ ] 作者贡献说明已准备。
- [ ] 版权协议、课题组长签名、保密审查证明等下载区材料已检查。
- [ ] 投稿系统账号、稿件文件、图表文件、补充材料已准备。
